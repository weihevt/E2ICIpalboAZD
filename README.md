## WEE1 inhibition delays resistance to CDK4/6 inhibitor and antiestrogen treatment in ER+ MCF7 cells
[12/24/2025] This repository contains the MATLAB (R2023a) code for the paper titled
["**WEE1 inhibition delays resistance to CDK4/6 inhibitor and antiestrogen treatment in ER+ MCF7 cells.**"](https://www.nature.com/articles/s41540-025-00604-z)

The **code** folder contains the functions used in the work.\
The **fig** folder contains the figures used in the paper.\
The simulation results from MATLAB and experimental data are first saved in the ./code/.xls/ folder, and then pasted into the Figs.prism file, which is used to generate the plots in the paper with GraphPad.Raw plots in the paper are generated by:\
Fig.1: manually.\
Fig.2A-D: Generated using GraphPad, with the corresponding data table saved in the file **Figs.prism**.\
Fig.2E: Western blot data plot.
Fig.2F: Western blot data plot.
Fig.3A-F: Generated using GraphPad, with the corresponding data table saved in the file **Figs.prism**.The raw plots are also generated by the function **E2ICIpalboAZD_15_doseresponse()**.\
Fig.4A: manually.\
Fig.4B: manually.\
Fig.5A-F: Generated using GraphPad, with the corresponding data table saved in the file **Figs.prism**. The raw plots are also generated by the function **E2ICIpalboAZD_26_plot()**.\
Fig.6A-F: Generated using GraphPad, with the corresponding data table saved in the file **Figs.prism**. The raw plots are also generated by the function **E2ICIpalboAZD_26_plot()**.\

**Functions:**\
***(1).E2ICIpalboAZD_01_main()***\
This function runs the simulation and produces plots using the optimized parameter set.\
***(2).E2ICIpalboAZD_02_readcellnum()***\
This function loads cell count data from the **_./data/coulter counter_** directory and saves it as a .mat file in **_./code/mat/Outcellnum.mat_** for use in model parameter fitting.\
***(3).E2ICIpalboAZD_03_readdata_western()***\
This function loads western blot data from the **_./data/western blot_** directory and saves it as a .mat file in **_./code/mat/Outprotein.mat_** for use in model parameter fitting.\
***(4).E2ICIpalboAZD_04_par()***\
This function provides the parameter values used in the ODE model, corresponding to the Supplementary Table 2 in the paper.\
***(5).E2ICIpalboAZD_05_modelpar()***\
This function generates the necessary given parameters for the model simulation and lists the variables defined in the model.\
***(6).E2ICIpalboAZD_06_test()***\
This function estimates model parameters based on experimental data.\
***(7).E2ICIpalboAZD_07_evlcost()***\
This function is for parameter estimation during optimization. It calls the function **_E2ICIpalboAZD_08_cost()_**.\
***(8).E2ICIpalboAZD_08_cost()***\
This function calculates the cost (error) between model simulation and experimental data.\
***(9).E2ICIpalboAZD_09_onesteadystate()***\
This function checks whether two initial values, zero and a positive one result in the same steady state rather than distinct ones.\
***(10).E2ICIpalboAZD_10_mutation()***\
This self-modifying function is used as the mutation function within the genetic algorithm (ga).\
***(11).E2ICIpalboAZD_11_gaoutput()***\
This function is output function in the GA algorithm saves the parameter cohort to **_./code/mat/opmat.mat_**.\
***(12).E2ICIpalboAZD_12_loaddata()***\
This function loads the data from the **_./code/mat/_** folder needed for parameter calibration in the model.\
***(13).E2ICIpalboAZD_13_model()***\
This function implements the ODE model, with its equations detailed in Supplementary Note 2 of the paper.\
***(14).E2ICIpalboAZD_14_assignval()***\
This function maps variables in the mathematical model to their corresponding counterparts.\
***(15).E2ICIpalboAZD_15_doseresponse()***\
This function generates the raw plots for Fig. 3A–F.\
***(16).E2ICIpalboAZD_16_barplot()***\
This function generates a bar plot.\
***(17).E2ICIpalboAZD_17_changemedia()***\
This function sets the time points when the media is changed. In the experiment, the media is replaced every 3–4 days.\
***(20).E2ICIpalboAZD_20_processdata()***\
This function processes the experimental data.\
***(21).E2ICIpalboAZD_21_replating()***\
This function resets the initial cell number each month, simulating the replating done in the experiment.\
***(22).E2ICIpalboAZD_22_minimumAZD_2treat()***\
This function generates the treatment options shown in Fig.7A-C, selecting from three possible treatment choices.\
***(23).E2ICIpalboAZD_23_minimumAZD_3treat()***\
This function generates the treatment options shown in Fig. 7E–F, selecting from three possible treatment choices.\
***(24).E2ICIpalboAZD_24_genparcohort()***\
This function generates **_./code/mat/opmat.mat_**, the first step is to create the parameter cohort.\
***(25).E2ICIpalboAZD_25_findcohort()***\
This function generates the parameter cohort and is called by **_E2ICIpalboAZD_24_genparcohort()_**.\
***(26).E2ICIpalboAZD_26_plot()***\
This function plots the simulation results from the parameter cohort. These are the results included in the paper.
***(27).E2ICIpalboAZD_27_plotalterpro()***\
This function plots the proliferation simulation results for the alternating treatment.\
***(28).E2ICIpalboAZD_28_plotpro()***\
This function plots the simulation results alongside the experimental data.\
***(29).E2ICIpalboAZD_29_sim()***\
This function simulates various treatment conditions.\
***(30).E2ICIpalboAZD_30_result()***\
This function generates simulation results for each parameter set in the parameter cohort. It is called by **_E2ICIpalboAZD_32_simparcohort()_**.\
***(31).E2ICIpalboAZD_31_simalternation()***\
This function simulates alternating treatments.\
***(32).E2ICIpalboAZD_32_simparcohort()***\
This function simulates all parameter sets stored in **_./code/mat/opmat.mat_**.\
***(33).E2ICIpalboAZD_33_simplotcheck()***\
This function plots the dynamics of certain compounds in the model for inspection.\
